# talkomatic

A real-time group chat web application inspired by the original Talkomatic from the 1970s.

## Features

- Real-time messaging
- Usernames and color theming
- Single chat room on local IP
- Simple and intuitive interface

## Raku Modules Used

- `zef install cro`
- `zef install --/test Cro::HTTP Cro::WebSocket`

## Silly Feature Ideas

- Away messages
- Sound effects
- Message reactions
- TLS termination
- Markdown support in textareas
- Tab notifications
